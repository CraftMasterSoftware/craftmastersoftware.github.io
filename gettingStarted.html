Welcome to CraftMaster. Lets get started!

Worlds Workspace

Select each of the 3 main Workspaces of CraftMaster (Worlds, 3D Viewer, and Downloads).

Select which World is active, and manage Worlds and Maps.

Choose Dimension, update current Map, export and delete current World.

View and place available 3D Objects.

Show Sign on the Map, teleport to Sign, and extract or delete a volume defined by 2 Signs.

Show Structure on the Map, teleport to Structure, and extract or delete a volume defined by a Structure Block.

Complete view of the entire World Map.

Current coordinates the cursor is hovering over, and Sign name if that block contains a Sign.

  • Click and drag with the mouse or use arrow keys to pan, scroll with the mouse to zoom in and out.

  • Update the current Map to see the latest state of the Minecraft World.

  • Use Minecraft to place Signs in the World to identify landmarks and define volumes.

  • 2 Signs with the same name define a volume (upper inside corner of one to lower inside corner of other).

  • Grey rectangles are volumes defined by Signs, white rectangles are volumes defined by Structure Blocks.

  • Structures can be placed in World using 3D Objects tool — use , and . keys to rotate before placing.

  • Structures can be extracted from World (copied into CraftMaster library) using Signs or Structure Blocks.

  • Volumes can be deleted from World using Signs or Structure Blocks.

3D Workspace

Select each of the 3 main Workspaces of CraftMaster (Worlds, 3D Viewer, and Downloads).

Duplicate, delete, and export 3D Object. Share on Sketchfab, Create Structure Block to place in current World.

Adjust the number of colors in the 3D Object and Blur the color to adjust the color mapping algorithm.

Customize the appearance of the 3D Object by selecting different Minecraft blocks for

each color represented. Save the blocks picked as a Palette

Select which 3D Object is viewed, import and organize 3D Objects.

Enter new X,Y, or Z values to change scale of 3D Object.

Update 3D Object after changing scale and/or color parameters.

Define, apply, set default, and manage Palettes of Minecraft blocks.

  • Navigate within the 3D Viewer as follows:

    • Click and drag with the mouse to rotate, and scroll with the mouse to zoom in/out.

    • Change the camera view using the arrow keys (←, ↑, →, and ↓).

    • Spin the camera view using bracket keys ([ and ]).

    • Move the camera using Minecraft navigation keys (w, s, a, d, left shift, and space).

    • Hit ESC to return to the default view.

  • Rotate the 3D Object in 90 degree increments for placement in Minecraft Worlds using , and . keys.

  • 3D Object will be placed relative to the Structure Block, which is represented by the bright pink cube.

  • Sketchfab upload requires authenticated Sketchfab account — sign–up at www.sketchfab.com..

  • "Save Structure to Current World" command places Structure Block in the currently active World.

  • Select desired export file format by typing file extension (.OBJ, .STL, .CMO, etc.).

Get Content

Select each of the 3 main Workspaces of CraftMaster (Worlds, 3D Viewer, and Downloads).

Import 3D models, schematics, or CraftMaster Objects from local disk, or choose one of several sites to download 3D models, schematics or Objects.

Back, Forward, Reload, and Home buttons.

  • Listed below are the options for acquiring 3D objects for use with CraftMaster:

    • Import 3D Objects (.CMO), 3D models (.OBJ, .KMZ, .DAE) and schematics from local storage.

    • Import 3D Objects (.CMO) from CraftMaster’s content page (not yet active).

    • Import 3D models (.KMZ, .DAE) from 3D Warehouse.

    • Import schematics from MinecraftStuff.net and Minecraft-Schematics.com.

  • IMPORTANT: When downloading 3D models from 3D Warehouse, please note that CraftMaster currently does not support the .SKP file format. So, after clicking Download, please choose THE LAST option, which is typically either KMZ or Collada (.DAE).

  • To use Minecraft-Schematics.com, you need to register for an account.

  • Imported or downloaded content will appear in 3D Viewer.
